# Complaint Management System

A web-based Complaint Management System developed using **Python, and SQLite** to simplify complaint registration and management. The system allows users to submit complaints and enables administrators to efficiently manage and track them.

---

## 📌 Project Overview

This project provides an online platform where users can register complaints, upload audio complaints, and track their submissions. The administrator can manage user complaints through a secure admin dashboard.

---

## ✨ Features

- User Registration & Login
- Admin Login
- Submit Text Complaints
- Upload Audio Complaints
- OTP Verification
- Complaint Management Dashboard
- Secure Authentication
- Simple and User-Friendly Interface

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/Pujithacheguri/Complaint-Management-System.git
```

2. Move into the project folder

```bash
cd Complaint-Management-System
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the project

```bash
python manage.py runserver
```

5. Open your browser and visit

```
http://127.0.0.1:8000/
```
